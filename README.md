# Game of Life
Conway's "Game of Life" using React. The app simulates cellular automation. This is based on [freeCodeCamp's tutorial](https://www.youtube.com/watch?v=PM0_Er3SvFQ).

## How to Run
```bash
npm install
yarn start
```
